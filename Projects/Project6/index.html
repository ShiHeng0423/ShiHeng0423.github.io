Software Engineering Project 1

About the Game

Defend the space base as a customizable spaceship! Survive as long as you can while battling multiple enemy types and bosses.

Game Controls

Keyboard:

  • WASD: Move spaceship
  • Left / Right Arrow: Navigate through powerups
  • Enter: Use selected powerup
  • B: Enter shop when near the base
  • Spacebar: Shoot
  • ESC: Pause the game

Mouse:

  • Click to interact in shop or menus

Contributions

  • Implemented Weapons & Bullets (Basic & Special bullets, bullet sprites, sfx, collision behavior, weapon types)
  • Developed Sound Manager (Music, BGM & SFX settings)
  • Implemented Game Manager
  • Created 1-time use item logic and UI
  • Built Debugging features
  • Developed Sound Setting UI
