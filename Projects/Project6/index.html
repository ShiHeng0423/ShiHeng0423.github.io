About the Game

Defend the space base as a customizable spaceship! Survive for as long as possible while battling enemy waves and bosses.

Game Controls

Keyboard

  • WASD: Move spaceship
  • Left / Right Arrow: Switch powerups
  • Enter: Use selected powerup
  • B: Enter shop when near base
  • Spacebar: Shoot
  • ESC: Pause

Mouse

  • Click to interact with shop or menus

My Contributions

  • Implemented all weapon and bullet systems (basic, special, SFX, collision behaviour, weapon types).
  • Developed the Sound Manager for BGM and SFX settings.
  • Implemented the overall Game Manager system.
  • Created logic and UI for one-time use items.
  • Built a set of debugging features for development and testing.
  • Developed full sound settings UI.
