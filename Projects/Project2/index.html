Unreal Project – USA Summer Class

About the Game

This Unreal Engine project was developed during a USA summer class. It features a handcrafted first-person 3D level with exploration-focused gameplay.

  • First-person, single-player experience
  • Designed as a 7–10 minute run
  • Fully handcrafted level layout

Controls

  • Move: WASD
  • Rotate: Mouse movement
  • Interact: F
  • Note: Game SFX may be loud — consider lowering volume
