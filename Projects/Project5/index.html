Software Engineering Project 2 (Year 1)

About the Game

MonstroStriko is a side-scrolling Metroidvania-style action-adventure game where players explore a dynamic dungeon filled with enemies, hazards, and secrets.

Players wield an intuitive weapon and combo system as they progress through interconnected levels, unlock new abilities, and overcome increasingly challenging foes. Each encounter—from minor creatures to formidable bosses—demands precision, timing, and adaptability.

The game features transitional animations, a climactic boss encounter, environmental interactions, and progression mechanics that reward exploration and skillful play.

Genre: Side-Scrolling • Action Adventure • Metroidvania

Game Controls

Keyboard:

  • A / D: Move Left & Right
  • I: Interact with NPCs
  • Space: Jump
  • Tab: Open Inventory
  • Esc: Pause Game

Mouse:

  • Left Click: Equip / Select

Contributions

  • Implemented full player control system, including movement, combat, interaction, inventory, and crafting.
  • Developed the Audio Manager to handle game-wide SFX and music.
  • Built the camera system with follow behavior, screen shake, and cinematic effects.
  • Optimized and implemented collision logic for players, enemies, projectiles, bosses, NPCs, and interactables.
  • Designed and programmed all major UI components, including Main Menu, Inventory, Armor/Weapon UI, Pause Menu, and HUD elements.
  • Created Armor and Weapon systems with rarity, stats, set bonuses, and passives (e.g., Lifesteal).
  • Developed a modular Status Effect System with UI indicators.
  • Built extensive debugging tools such as teleport, invincibility, max attack, and god-mode save file.
  • Implemented the Game Manager to coordinate core systems and ensure a single persistent player instance.
