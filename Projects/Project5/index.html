Software Engineering Project 2 (Year 1)

About the Game

MonstroStriko is a side-scrolling, Metroidvania-style action-adventure game where players explore a dynamic dungeon filled with enemies, hazards, and secrets.

Players wield a responsive weapon and combo system while progressing through interconnected levels, unlocking new abilities, and facing increasingly challenging foes. From minor creatures to major bosses, encounters demand precision, timing, and adaptability.

The game features transitional animations, a climactic boss battle, environmental interactions, and progression systems that reward exploration and skillful play.

Genre: Side-Scrolling • Action Adventure • Metroidvania

Game Controls

Keyboard

  • A / D: Move left & right
  • I: Interact with NPCs
  • Space: Jump
  • Tab: Open inventory
  • Esc: Pause game

Mouse

  • Left Click: Equip / Select

My Contributions

  • Implemented the full player control system: movement, combat, interaction, inventory, and crafting.
  • Developed the Audio Manager to handle game-wide sound effects and music.
  • Built the camera system, including follow behaviour, screen shake, and cinematic effects.
  • Optimised and implemented collision logic for: players, enemies, projectiles, bosses, NPCs, and interactable objects.
  • Designed and programmed major UI components: Main Menu, Inventory, Armor/Weapon UI, Pause Menu, and HUD elements.
  • Created the Armor and Weapon systems with rarity, stats, set bonuses, and passives (e.g., Lifesteal).
  • Developed a modular Status Effect System with in-game UI indicators.
  • Built debugging tools such as teleport, invincibility, max attack, and a “god-mode” save file for faster iteration.
  • Implemented the Game Manager to coordinate core systems and maintain a single persistent player instance.
