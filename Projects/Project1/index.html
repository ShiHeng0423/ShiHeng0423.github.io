Unity Project at USA Summer Class

About the Game

This is a 2D single-player, top-down action game built during a USA summer Unity class. The game features a procedurally generated dungeon-style level with short, replayable runs.

  • Top-down 2D gameplay.
  • Procedurally generated level layout.
  • Designed for ~5 minute runs per playthrough.

Controls

  • Move: WASD
  • Aim / Rotate: Mouse movement
  • Shoot: Left Mouse Button
  • F1: Restart the level
  • F2: Zoom out to show the whole level
  • F3: Toggle "God Mode"
