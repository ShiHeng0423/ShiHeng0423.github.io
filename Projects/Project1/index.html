Unity Project at USA summer class

About the Game

2D single player, top down procedurally generated level.

About 5 minutes long for a single run.

Controls

Move: WASD

Aim / Rotate: Mouse Movement

Shoot: Left Mouse Button

F1: Restart the level

F2: Zoom out to show whole level

F3: Toggle "God Mode"
