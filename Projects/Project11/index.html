Studio Project 3 — Road to Redemption

Overview

Road to Redemption is a 2D survival game inspired by Death’s Gambit. Players progress through procedurally generated levels, defeat enemies, collect upgrades, and race against time to clear the game as quickly as possible while achieving the highest score.

Key Features

  • Randomly generated multi-room maps.
  • Four enemy types with distinct AI behaviours.
  • Variety of projectile attacks.
  • Loot drops and upgrade system.
  • Leaderboard with time-based scoring.
  • Main Menu, Pause Menu, Win/Lose screens.
  • Sound effects and background music.

Controls

  • A / D – Move left & right
  • Space – Jump
  • O – Open doors
  • E – Shoot projectile
  • I – Open menu / Back
  • Enter – Select

Gameplay Systems

  • Random Map Generator: Multi-room layout constructed via linked-list rooms with door-based progression.
  • Player Movement: Walking, sliding, and crouching with tile-based collision.
  • Upgrades: Mini points → large points used to upgrade HP, damage, and reload speed.
  • Loot Types: Damage boost, HP regeneration, and upgrade points.
  • Projectiles: Fireball, Arrow, and Bullet, each with different behaviours.
  • Map Editor: CSV-based maps with tile IDs, collision flags, and visual variants.

  • Designed and implemented the random map generation.
