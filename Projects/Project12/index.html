Unity Project at USA summer class (Groupwork)

About the Game

This project simulates how humanistic AI behaves in a game environment.

Controls

Move: WASD

Shoot: Left Mouse Click

F1: Enemy is Omniscient

F2: Enemy detects player by View Cone

F3: Enemy detects player by Poisson Disc

F4: Enemy detects player by Circle Radius

F5: Toggle dotted path visibility to player

F6: Toggle enemy hearing

F7: Toggle Gaussian Randomness on View Cone Raycast

F8: Toggle Raycast Line Visibility (when Gaussian Randomness is enabled)

F9: Toggle Zoom Out (view entire map)

F10: Restart / Reset Level

Contributions

Implemented the shooting logic, including bullet firing sound and travel sound effects.

Developed sound emission logic and how sound is attenuated based on wall properties (hollow walls allow more sound to pass, solid walls less).

Ensured AI blackboard data is error-free when patrolling or exchanging information only with sighted AI.
