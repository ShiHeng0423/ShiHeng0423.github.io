Studio Project 1

About the Game

Studio Project 1 is an ASCII-based maze game where the entire maze layout is generated by reading characters from a text file. Each ASCII character represents a different tile or wall type, making the game highly flexible and easy to modify simply by editing a text file. This design allowed for rapid iteration and creative level building with zero art assets.

  • Enter – Confirm
