Studio Project 1 — ASCII Maze Game

Overview

Studio Project 1 is an ASCII-based maze game where the entire level layout is generated from a plain text file. Each character in the file represents walls, floor tiles, or other elements that form the maze environment.

This project focused on fundamentals such as file parsing, grid-based movement, and designing a fully playable game loop with no art assets — everything is built entirely from text.

Controls

Menu Navigation

  • Enter — Confirm Selection

Gameplay
