Studio Project 1

About the Game

Studio Project 1 is an ASCII-based maze game where the entire playable level is generated by reading characters directly from a text file. Each ASCII symbol corresponds to a tile, wall, or interactable object. This approach allowed rapid iteration, easy level editing, and a complete game built with zero art assets.

The project introduced foundational concepts such as file parsing, grid systems, movement handling, and basic game loop structure—all implemented purely in C++.
